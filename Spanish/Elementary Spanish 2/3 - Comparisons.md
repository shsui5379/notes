# Superlatives
- adjective meaning the maximum of a quality
- bueno -> mejor: good -> best
- malo -> peor: bad -> worst
- joven -> menor: young -> youngest
- viejo -> mayor: old -> oldest
- grande -> mayor: big -> biggest
- pequeño -> menor: small -> smallest
