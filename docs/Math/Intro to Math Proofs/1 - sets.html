Sets

Sets

  • A collection of elements
  • Noted by capital letters
  • Ways to define a set
    • List
      • S={1,2,3}S = \{1,2,3\}S={1,2,3}
      • Just listing all the elements out
    • Set builder notation
      • S={x∣x>0}S = \{ x | x > 0\}S={x∣x>0}
      • ∣|∣ and ::: means "such that"
      • {expression∣condition}\{ expression | condition \}{expression∣condition}
    • Descriptive definition: using a sentence/paragraph to describe the elements of the set
    • Interval Notation
      • [2,∞)[2, \infty)[2,∞)
      • Parenthesis means exclusive at the endpoint
        • Infinity is always exclusive
      • Square brackets mean inclusive at the endpoint
      • (start,end)(start, end)(start,end)

Important sets

  • Natural numbers: N={1,2,3,4,...}N = \{1, 2, 3, 4, ...\}N={1,2,3,4,...}
    • counting numbers
  • Integers: Z={0,±1,±2,±3,...}Z = \{0, \pm 1, \pm 2, \pm 3, ... \}Z={0,±1,±2,±3,...}
    • Z+Z^+Z+ are positive integers, while Z−Z^-Z− are negative integers
    • ZZZ is closed under addition and multiplication, meaning adding or multiplying 2 integers together will not result in something that's not an integer
      • if a,b∈Za,b \in Za,b∈Z, then a+b∈Za+b \in Za+b∈Z and a∗b∈Za*b \in Za∗b∈Z
        • ∈\in∈ means that something is an element of a set
      • Even integers = {2x∣x∈Z}\{2x | x \in Z\}{2x∣x∈Z}
      • Odd integers = {2x+1∣x∈Z}\{ 2x + 1 | x \in Z\}{2x+1∣x∈Z}
  • ∅\emptyset∅: empty set
  • Quotient of integers (rational numbers): Q={pq∣p,q∈Z,q≠0}Q = \{\frac{p}{q} | p, q \in Z, q \not ={0}\}Q={qp​∣p,q∈Z,q=0}
    • Repeating decimals are rational too
      • If the repeating part is x digits long, divide the repeating part by a number with x number of 9's
  • Real numbers: RRR

Unions

  • Union
    • A∪B={x∈A∨x∈B}A \cup B = \{x \in A \lor x \in B \}A∪B={x∈A∨x∈B}
    • A union between sets A and B is a set containing all elements from A and B
  • Intersect
    • A∩B={x∈A∧x∈B}A \cap B = \{x \in A \land x \in B\}A∩B={x∈A∧x∈B}
    • An intersect between sets A and B is a set of elements that are found in both A and B
  • Difference
    • A−B={x∈A∧x∉B}A - B = \{x \in A \land x \notin B\}A−B={x∈A∧x∈/B}
    • The difference between sets A and B is a set containing elements from A that aren't in B
